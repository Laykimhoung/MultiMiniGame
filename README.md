## Features

- Multiple mini-games bundled in one project.  
- Game 1: Guess The Word
- Game 2: Who Wants to Be a Millionair
- Game 3: Boss Attack
- Game 4: Breakout

---

## Technologies Used

- **Language:** C#  
- **Framework:** .NET 10  
- **IDE:** Visual Studio 2022  
- **Libraries:**  
  - [`LibVLCSharp`] for video/audio playback  
  - WinForms for UI
